# Cmpedu Downloader

机械工业出版社教育服务网资源下载，无需登录，无需教师权限，油猴脚本。

## 如何使用

例如，当您打开`http://www.cmpedu.com/books/book/12345.htm` 或 `http://www.cmpedu.com/ziyuans/ziyuan/12345.htm`等类似网页时，脚本程序会自动在页面上显示该资源的下载链接，点击下载按钮即可下载。
